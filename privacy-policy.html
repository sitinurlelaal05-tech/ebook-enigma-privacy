Privacy Policy

TL;DR (Too Long; Didn't Read):
Aplikasi Katalog Ebook Digital TIDAK mengumpulkan, menyimpan, atau membagikan data pribadi Anda sama sekali. Aplikasi ini 100% offline dan tidak melakukan koneksi internet.

1. Informasi yang Kami Kumpulkan

TIDAK ADA.

Aplikasi Katalog Ebook Digital tidak mengumpulkan informasi apapun dari pengguna, termasuk namun tidak terbatas pada:

  • Informasi pribadi (nama, email, nomor telepon, dll.)
  • Data lokasi
  • Device identifiers (IMEI, Android ID, dll.)
  • Data penggunaan aplikasi
  • Crash reports atau analytics
  • Cookies atau tracking technologies

2. Bagaimana Kami Menggunakan Informasi

TIDAK APPLICABLE.

Karena kami tidak mengumpulkan data apapun, maka tidak ada data yang digunakan, diproses, atau dibagikan.

3. Penyimpanan Data Lokal

Aplikasi ini menyimpan preferensi dan data berikut HANYA DI DEVICE ANDA (tidak dikirim ke server):

  • Preferensi tampilan: Pilihan tampilan Grid atau List
  • Bookmark: Halaman terakhir dibaca untuk setiap ebook
  • ID ebook terakhir dibaca: Untuk quick access

Data ini disimpan menggunakan Android DataStore dan tetap berada di device Anda. Data ini tidak dapat diakses oleh aplikasi lain dan akan terhapus jika Anda uninstall aplikasi.

4. Koneksi Internet

TIDAK ADA KONEKSI INTERNET.

Aplikasi ini dirancang untuk berfungsi 100% offline. Aplikasi tidak memerlukan dan tidak meminta permission INTERNET. Semua konten (ebook PDF) tersimpan di dalam aplikasi dan dapat diakses tanpa koneksi internet.

5. Iklan dan Analytics

TIDAK ADA IKLAN. TIDAK ADA ANALYTICS.

Aplikasi ini tidak menampilkan iklan (AdMob atau platform iklan lainnya) dan tidak menggunakan analytics tools (Google Analytics, Firebase Analytics, atau sejenisnya).

6. Notifikasi Push

TIDAK ADA.

Aplikasi ini tidak menggunakan Firebase Cloud Messaging (FCM) atau layanan push notification lainnya.

7. Third-Party Services

TIDAK ADA.

Aplikasi ini tidak mengintegrasikan third-party services apapun yang dapat mengumpulkan data pengguna.

8. Permissions

Aplikasi ini TIDAK meminta permission apapun, termasuk:

  • ❌ INTERNET - Tidak ada koneksi internet
  • ❌ READ_EXTERNAL_STORAGE - Semua konten dari assets internal
  • ❌ WRITE_EXTERNAL_STORAGE - Tidak menyimpan ke external storage
  • ❌ ACCESS_FINE_LOCATION - Tidak butuh lokasi
  • ❌ CAMERA - Tidak ada fitur camera
  • ❌ CONTACTS - Tidak akses contacts

9. Keamanan Data

Karena tidak ada data yang dikumpulkan atau ditransmisikan, maka tidak ada risiko keamanan data pribadi. Semua data yang disimpan lokal di device Anda dilindungi oleh security sandbox Android dan hanya dapat diakses oleh aplikasi ini.

10. Hak Anda

Karena kami tidak mengumpulkan data apapun, maka tidak ada data yang perlu diakses, diubah, atau dihapus. Namun, Anda memiliki kontrol penuh atas data lokal dengan cara:

  • Hapus preferensi: Clear data aplikasi di Settings > Apps
  • Hapus semua data: Uninstall aplikasi

11. Children's Privacy

Aplikasi ini aman untuk semua umur termasuk anak-anak di bawah 13 tahun karena tidak mengumpulkan data apapun. Kami comply dengan Children's Online Privacy Protection Act (COPPA).

Jika ada perubahan pada privacy policy ini, kami akan mengupdate halaman ini dan mengupdate "Last Updated" date di atas. Perubahan signifikan juga akan diumumkan melalui update aplikasi.

13. Compliance
